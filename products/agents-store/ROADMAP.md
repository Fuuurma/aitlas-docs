# Aitlas Agents - Future Roadmap

**Status:** Planning Phase  
**Last Updated:** 2026-03-08

---

## Overview

Agents are specialized AI assistants with domain-specific knowledge, tools, and workflows. Each agent has a unique system prompt, tool access, and behavior profile.

## Agent Architecture

### Core Components

```
agent/
├── system-prompt.md      # Agent personality & behavior
├── tools.yaml            # Available MCP tools
├── workflows/            # Pre-defined workflows
├── memory/               # Agent-specific memory
└── config.yaml           # Agent configuration
```

### System Prompt Template

```markdown
# Agent: {name}

## Identity
You are {name}, a specialized {role}.

## Capabilities
- {capability_1}
- {capability_2}
- {capability_3}

## Tools
You have access to: {tool_list}

## Behavior
- {behavior_guideline_1}
- {behavior_guideline_2}

## Constraints
- {constraint_1}
- {constraint_2}

## Example Interactions
{examples}
```

## Creating Custom Agents

Users can create custom agents with:

```yaml
# my-agent.yaml
name: "f.my-agent"
role: "Custom Role"
capabilities:
  - "Capability 1"
  - "Capability 2"
tools:
  - f.scrape
  - f.vault
behavior:
  - "Always cite sources"
  - "Be concise"
constraints:
  - "Don't make up facts"
```
