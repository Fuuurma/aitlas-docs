# Aitlas Research

Categorized research for Aitlas products.

## Structure

| Folder | Description |
|--------|-------------|
| [actions/](actions/) | Agent frameworks, orchestration, MCP integrations |
| [agents/](agents/) | Skills stores, agent marketplaces |
| [nexus/](nexus/) | Workflow platforms, orchestration runtimes |
| [nova/](nova/) | Multi-provider UIs, coding agents |

## Categories

### Actions (20 files)
Agent frameworks and orchestration tools:
- AutoGen, CrewAI, Mastra, MetaGPT (multi-agent frameworks)
- OpenSandbox, OSWorld (sandbox environments)
- MCP, TaskingAI, Swam (protocols & frameworks)
- And more...

### Agents (2 files)
Skills stores and agent marketplaces:
- Antigravity Skills
- Sanyuan Skills

### Nexus (5 files)
Workflow orchestration platforms:
- Coze, Dify (no-code workflow platforms)
- Trigger.dev (background execution)
- AionUi (desktop UI alternative)
- Floop Analysis (runtime analysis)

### Nova (7 files)
Multi-provider UI and coding agents:
- T3Code (Chat UI)
- Codex, OpenCode (coding agents)
- Multi-provider integration
- Nexus hybrid model

## Contributing

When adding new research:
1. Choose the correct category folder
2. Use `_RESEARCH.md` or `_INTEGRATION.md` suffix
3. Include: status, stars, category, use case
