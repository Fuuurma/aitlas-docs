## Key Features

| Feature | Description |
|---------|-------------|
| **Setup UI** | Web dashboard for config |
| **Guided Onboarding** | Step-by-step wizard |
| **Gateway Manager** | Spawns, monitors, restarts agents |
| **Watchdog** | Crash detection, auto-recovery |
| **Channel Orchestration** | Telegram/Discord bots |
| **Webhooks** | Named endpoints with transforms |
| **Google Workspace** | OAuth for Gmail, Calendar, Drive |
| **File Explorer** | Browser-based workspace |
| **Git Sync** | Auto-commit workspace |
| **Version Management** | In-place updates |
